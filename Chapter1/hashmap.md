# 1.5 哈希表

哈希表：通过 哈希函数 计算数据存储位置的 一直数据结构。
哈希表 这种数据结构支持的主要操作：
1. insert(key, value) # 快速插入
2. get(key) # 快速查找
3. delete(key) # 直接删除

#### 一、常用的 几种哈希算法
基本思想： 根据 key 直接寻址
![image.png](https://rnd-isourceb.huawei.com/images/NJ/20200321/4db50c0f-9e7e-463d-a27b-78f7e8502654/image.png)

- 优点：插入/删除/查找 都超快！
- 缺点：消耗大量内存，且大量空间可能被浪费，而且能处理的key类型极其有限。

key 怎么进行合适的处理？ 引入 hash函数
1. 除法哈希
2. 乘法哈希
3. 全域哈希

出现 hash冲突 之后怎么解决？
1. 拉链法
2. 线性扩展/二次扩展/多层哈希/more

#### 二、Python 对哈希的应用

> python 中的 字典/集合 都是通过哈希表实现的。
